# [Design Spec] 기업 신용위험 모니터링 시스템 (Downside Risk Monitoring System)

이 설계 문서는 20년 경력 베테랑 심사역의 비판적 시각을 반영한 기업 신용위험 모니터링 시스템을 단일 HTML 파일로 구현하기 위한 디자인 스펙입니다.

## 1. 개요 및 설계 목표
- **목표**: 철강, 화학 등 주요 원자재 시클리컬 기업의 핵심 선행 지표(롤마진, 스프레드, 재고 회전기일 등)를 실시간으로 모니터링하고 분석할 수 있는 단일 HTML 기반의 프로페셔널 웹 애플리케이션을 구축합니다.
- **주요 대상**: 포스코(철강), 현대제철(철강), 롯데케미칼(석유화학).
- **디자인 철학**: 복잡한 빌드 도구 없이 단일 파일로 동작하되, 금융 단말기처럼 높은 신뢰도와 풍부한 인터랙션을 제공하며 가독성 높은 UI/UX를 구현합니다.

## 2. 레이아웃 아키텍처 (C안: 사이드바 탭 포털 구조)
- **전체 화면 분할 (100vh)**:
  - **좌측 사이드바 (Width: 260px)**: 
    - 브랜드 로고 및 타이틀
    - 메인 탭 메뉴 (대시보드 홈, 신용 위험 심사, 시뮬레이터 설명)
    - 실시간 **Watchlist**: 등급별(Red/Yellow/Green) 현재 모니터링 기업 요약 목록
    - 퀵 매크로 온트랙 지표(철강 BEP, 에틸렌 BEP 기준선 상시 노출)
  - **우측 메인 콘텐츠 (Flex: 1, Overflow-y: auto)**:
    - **대시보드 홈 탭**: 전체 기업 현황 그리드 카드, 핵심 리스크 건수 카운터, 업종별 동향 요약.
    - **신용 위험 심사 탭**: 선택된 기업의 상세 신용 보고서.
      - 헤더: 기업명/업종/리스크 등급 배지
      - 주력 부문 판결 결과 (철강업종 한정 노출)
      - 리스크 트리거 요인 (적색/황색 강조 박스)
      - 킬러 스프레드 게이지 바 (BEP 대비 현재 마진 시각화)
      - 실시간 변수 시뮬레이터 (정량 데이터를 조절할 수 있는 Slider UI)
      - 상세 정량/정성 리스크 리스트
      - 심사역 종합 의견 란 (텍스트 편집기 및 저장 기능)
      - 보고서 출력 및 PDF 다운로드 기능 버튼

## 3. 핵심 기능 설계
### A. 실시간 데이터 시뮬레이터 및 위험 등급 연동
- 사용자가 상세 분석 화면의 슬라이더를 통해 원재료 가격이나 스프레드를 변경하면 다음이 실시간 계산됩니다.
  - **롤마진 및 스프레드 BEP Gap**: (현재 값 - BEP 기준치)
  - **위험 등급 자동 재산출**:
    - **Red**: 롤마진 역갭이 임계치를 초과하거나 재고 회전기일 증가폭이 극도로 큰 경우 (예: 롯데케미칼 스프레드 $200 이하, 재고회전 증가 10일 이상)
    - **Yellow**: 마진이 BEP 하단에 걸쳐 있거나 재고 회전기일이 소폭 증가한 경우 (예: 롤마진 $250~300 사이, 재고회전 증가 5~9일)
    - **Green**: 모든 지표가 BEP를 상회하고 재고 회전속도가 안정적인 경우 (예: 롤마진 $300 초과, 재고회전 증가 5일 미만)
- 슬라이더 이동 시 화면 상의 리스크 등급 배지, 상단 테두리 보더 색상, 게이지 바 차트가 즉각 애니메이션과 함께 업데이트됩니다.

### B. 로컬 상태 관리 (LocalStorage)
- 사용자가 조정한 수치 및 개별 메모(심사역 한줄평)는 `credit_risk_sim_data` 키로 브라우저 LocalStorage에 실시간 저장됩니다.
- 새로고침 시 이전에 수정한 시뮬레이션 상태가 그대로 유지됩니다.
- **초기화(Reset) 기능**: "기본값 복원" 버튼을 클릭 시 LocalStorage를 초기화하고 베테랑 심사역의 원본 2026-Q2 데이터베이스로 복구됩니다.

### C. 시각적 요소 및 차트
- 외부 무거운 라이브러리 대신, **Pure CSS & SVG 게이지 바**를 사용하여 성능을 극대화합니다.
- BEP(손익분기점) 대비 현재 스프레드가 가리키는 위치를 보여주는 눈금선과 바 차트를 구현합니다.
- 색상 체계:
  - Slate Dark Mode 배경 (#0f172a / #1e293b / #334155)
  - Red (위험): #ef4444
  - Yellow (주의): #f59e0b
  - Green (정상): #10b981
  - Accent (정보/포커스): #38bdf8

### D. 보고서 인쇄 및 PDF 저장
- CSS `@media print` 쿼리를 설정하여 인쇄 시 사이드바와 시뮬레이션 슬라이더 조절기 등 UI 불필요 요소를 자동으로 제거합니다.
- 실제 인쇄되는 보고서(A4 규격)는 정갈한 서체와 깔끔한 테두리로 출력되도록 리포트 스타일링을 완벽하게 적용합니다.

## 4. 데이터베이스 구조 (creditDB)
- **포스코**: 철강 / 기준값: 롤마진 $257, 재고증가 7일 (Yellow)
- **현대제철**: 철강 / 기준값: 스프레드 32.5만원, 롤마진 $260, 재고증가 8일 (Yellow)
- **롯데케미칼**: 석유화학 / 기준값: 스프레드 $200, 재고증가 12일 (Red)
- *추가 검증 조건식 규칙 정의*

## 5. 검증 계획
- **수동 검증**: 
  - 각 기업의 슬라이더를 조정하여 등급이 실시간으로 변하는지 확인 (Red -> Yellow -> Green)
  - 새로고침 시 데이터 유지 및 Reset 시 복원 정상 동작 확인
  - PDF 인쇄 미리보기 화면에서 인쇄 최적화 포맷 작동 여부 검사
